Sona Hospitals - Patient Meal Planning Portal

This is a lightweight static scaffold for the Sona Hospitals Patient Meal Planning Portal. It implements the visual layout, theme, and front-end interactions described in the provided specification.

Files created:
- index.html — Homepage with header, hero, stats, services, timeline, doctor profile, testimonials, booking form, blogs, footer
- styles.css — Theme and layout styles using the provided color palette
- script.js — Minimal UI behaviors and form handling (frontend only)

Quick start
1. Open `index.html` in your browser (double-click or use a local server).

Optional (recommended for development)
- Serve with a simple static server:

  python3 -m http.server 8000

Visit http://localhost:8000 to preview.
